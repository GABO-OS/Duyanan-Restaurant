Description:
  This module handles all functionalities related to customer accounts on
  the Duyanan Restaurant online ordering system. It covers account creation,
  secure login, and profile maintenance. A valid account is required before
  a customer can place online orders.

FEATURE 1.1 - USER REGISTRATION
--------------------------------------------------------------------------------

TEST CASE 1.1.1
  Test Case ID   : TC-UAM-001
  Requirement ID : REQ-UAM-001
  Test Scenario  : Verify that a user can register a new customer account with valid details.
  User Story     :
    As a new customer,
    I want to register an account using my unique personal details,
    So that my credentials are saved in the system and I can access online ordering.
  Acceptance Criteria | Successful Registration:
    Given I am on the registration page
    When I fill in all required fields (first name, last name, unique email, phone, address, and password) and click "Register"
    Then my account should be created successfully in the database
    And I should see a success popup message: "Registration Successful!"
    And I should be redirected to the login page when the popup is dismissed
  Preconditions  : User is on the registration page; the email is not yet registered in the system
  Test Data      :
    First Name   : Maria
    Last Name    : Santos
    Email        : maria.santos@gmail.com
    Phone        : 09123456789
    Address      : 123 Mango Street, Cebu City
    Password     : Maria@1234
  Test Steps     :
    1. Navigate to the registration page
    2. Enter "Maria" in the First Name field
    3. Enter "Santos" in the Last Name field
    4. Enter "maria.santos@gmail.com" in the Email field
    5. Enter "09123456789" in the Phone field
    6. Enter "123 Mango Street, Cebu City" in the Address field
    7. Enter "Maria@1234" in the Password field
    8. Click the "Register" button
  Expected Result:
    - Account is created successfully in the database
    - A success SweetAlert2 popup displays: "Registration Successful! Your account has been created. Please log in."
    - Dismissing the alert redirects the user to the login page
